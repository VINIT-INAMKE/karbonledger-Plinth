<interfaces>
<!-- Error code format from established validators (ConfigHolder, ProjectPolicy) -->
Error registry pattern (from ConfigHolder.hs / ProjectPolicy.hs):
```haskell
{- ======================================================================
   ERROR CODE REGISTRY - [Validator Name] Validator
   ======================================================================

   CODE - Short description
          Cause: Why this error occurs
          Fix: How to resolve

   ======================================================================
-}
```

Error code format: 3-letter prefix + 3-digit number
- Existing: CHE (ConfigHolder), DGE (DaoGovernance), PVE (ProjectVault), PPE (ProjectPolicy), CPE (CotPolicy)
- New (per user discretion): CEE (CetPolicy), UVE (UserVault), MKE (Marketplace), INE (IdentificationNft)

Usage pattern:
```haskell
P.traceError "CEE000"           -- fatal error
P.traceIfFalse "CEE002" condition  -- conditional check
```
</interfaces>
